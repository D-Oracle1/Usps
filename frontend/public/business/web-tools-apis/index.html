                Web Tools APIs Shut Down January 25, 2026

                Migrate NOW to USPS APIs

                The Web Tools API platform shuts down on Monday, January 25, 2026. All Web Tools API users must switch now to USPS APIs.

                • Act Now! Take the first step: Sign up for USPS APIs.
                • See resources and submit support tickets below.
                • Get live help in free, weekly migration support webinars.

                Sign Up for USPS APIs

                How to Switch to USPS APIs

                I'm a Web Tools API Developer

                1. Sign up for USPS APIs.
                2. Use the API mapping spreadsheets to map your current Web Tools APIs to the corresponding USPS APIs.
                3. Refer to the help resources below and contact us if you need more help.
                4. Provide feedback and migration status in the migration feedback form.

                I use a third-party platform that uses Web Tools APIs

                • Call or email your service provider to confirm they are migrating to USPS APIs. Only register for the legacy Web Tools APIs if your vendor or platform requires it.

                Migrate to USPS APIs: Self-Serve Resources

                • USPS APIs Migration Onboarding Guide (PDF)
                • FAQs - Web Tools to USPS APIs (PDF)

                Mapping Web Tools APIs to USPS APIs

                Use our mapping resources to help you transition to the new APIs you need for your platform.

                Start with the clickable USPS API Functional Equivalents chart (PDF).

                Show More

                To use the USPS API Functional Equivalents chart:

                1. Find the Web Tools APIs you use.
                2. Click a Web Tools API name to get its Excel mapping spreadsheet.
                3. Each spreadsheet will show how the input/output fields in your legacy Web Tools APIs map to the new USPS APIs.
                4. Click the USPS API name to view the full specification in the USPS API Catalog.

                Direct links to the USPS API input/output field mapping spreadsheets:
